# APM
Android App performance monitoring tools

## lib_ui
UI模块，包括：UI渲染性能监控、UI适配、UI渲染优化

## lib_pfm
性能模块，包括：内存、耗电、安装包、启动 几个部分的监控优化

## lib_net
网络模块，包括：网络库实践应用，网络性能优化与监控

## lib_io
I/O模块，包括：磁盘读写方面的优化与监控

## lib_storage
存储模块，包括：各类存储方案实践、监控与优化

## lib_stat
统计模块，包括：统计上报方案实践

## lib_hook
Hook工具类

## lib_tool
通用工具类

## lib_tool_sw
SW方案适配辅助工具类
